What this page runs, precisely

The engine here is prototype/browser_core.py — the public demo core, fetched verbatim and run by CPython-in-WebAssembly. It shares the licensed engine's contract but is strictly weaker by design: five hypothesis families, a more conservative evidence rule. It refuses Catalan, factorials, and Tribonacci — which the licensed engine stamps (graded in eval/, reconciled in BATTERIES.md). What's identical in both: neither ever stamps what it cannot prove.
